# lemonsaur-inspired static site

Minimal static site scaffold you can deploy via GitHub Pages.

Quick steps to publish:

1. Create a GitHub repo (for a user page use `yourusername.github.io`).
2. Commit these files and push to `main`.
3. In GitHub repo Settings → Pages select `main` branch and `/ (root)`.
4. After a minute your site will be available at `https://yourusername.github.io/` (or the project URL).

Files:
- `index.html` — homepage
- `styles.css` — simple styles

Customize the text and links, then add posts or pages as needed.
